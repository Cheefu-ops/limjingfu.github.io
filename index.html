Built a sentiment classifier to identify positive and negative movie reviews. Included cosine similarity search to find reviews similar to unseen input text.

⚡ Dataset: Cornell Polarity (Bo Pang et al., 2002) + Rotten Tomatoes
Python NLTK TF-IDF Keras Scikit-Learn Cosine Similarity
