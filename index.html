What I've Built

Heart Disease Prediction D4AE · Data Analysis

Built an end-to-end machine learning pipeline to predict whether a patient has heart disease based on 14 clinical features. Compared three classifiers and identified the top predictors using feature importance analysis.

⚡ Logistic Regression AUC: 0.927 · KNN best accuracy: 91.8%
Python Scikit-Learn Pandas Seaborn Logistic Regression KNN Random Forest
Traffic Sign Recognition CNN CV Essentials · Computer Vision

Developed a Traffic Sign Recognition system to classify Singapore Stop and U-Turn signs using a 6-layer Convolutional Neural Network. Images were collected from Google Street View, augmented, and used to train a binary classifier. Extended with a sliding window object detection system for real-world images.

⚡ Validation accuracy: ~97–99%
Python TensorFlow / Keras OpenCV CNN ImageDataGenerator Google Colab
Movie Review Sentiment Analysis NLP Essentials · Natural Language Processing

Built a sentiment classifier to identify positive and negative movie reviews. Preprocessed 2,000 reviews using tokenisation, stopword removal, and lemmatisation, then extracted features with TF-IDF before training a Neural Network classifier. Included cosine similarity search to find reviews similar to unseen input text.

⚡ Dataset: Cornell Polarity (Bo Pang et al., 2002) + Rotten Tomatoes
Python NLTK TF-IDF Keras Scikit-Learn Cosine Similarity
